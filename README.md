# React Project (Task Tracker App)

Includes the react ui as well as JSON-server for our mock backend. Taught by [TravesyMedia](https://www.youtube.com/watch?v=w7ejDZ8SWv8).

## Usage

### Install dependencies

```
npm install
```

### Run React dev server (http://localhost:3000)

```
npm start
```

### Run the JSON server (http://localhost:5000)

```
npm run server
```

### To build for production

```
npm run build
```
